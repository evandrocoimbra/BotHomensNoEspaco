# BotHomensNoEspaco
Bot do Telegram que consulta quem são as pessoas no espaço atualmente.

Primeiro você deve obter uma chave da API do Telegram, escrevendo para o @BotFather. 
Após a mensagem inicial, digite /newbot. Você deverá informar então o nome do seu bot e o usuário que irá utilizar (deve terminar com a palavra bot).
Após receber a chave da API, você deve criar uma variável de ambiente no S.O., contendo essa chave. No caso utilizado aqui, criei a variável "SPACE_BOT_TOKEN".
Para interagir com o bot, digite /people.
